# Pull Log

Records each local `git pull --rebase --autostash` triggered by the crontab (2:45 AM nightly, 8:00 AM Sunday). Append-only — never overwrite.

Each entry shows: when the pull ran, which remote commit was fetched, and a `git diff --stat` of what changed.
